鼠标操作

  • 左键拖拽旋转视角
  • 右键拖拽平移视角
  • 滚轮缩放视角
  • 双击场景聚焦到双击位置的场景点
  • 拖拽文件导入模型到场景

键盘快捷键

  • Space播放 / 暂停相机动画
  • F聚焦当前选中模型
  • Ctrl + F在当前视角和时间新增相机关键帧；若后方无关键帧则时间轴前进 1 秒
  • ` / ~切换清屏模式
  • 1 / 2 / 3切换位移 / 旋转 / 缩放 gizmo；重复按同键可关闭
  • X相机回正
  • ?打开操作提示
  • Esc关闭操作提示或导出窗口

相机移动

  • W / A / S / D前后左右移动
  • 方向键前后左右移动
  • PageUp / PageDown上升 / 下降
  • Shift加速移动
