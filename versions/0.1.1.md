
# 0.1.1

## Added

- Added draggable width resizing for the model list and settings panel.
- Added an ONNX clip overflow indicator near the clip end when animation playback exceeds the clip duration.

## Changed

- Refined narrow-width layout behavior so the right settings panel remains usable when compressed.
- Adjusted transform rows, range controls, and numeric inputs so they align better within limited panel width.
- Tightened the compact layout for the model list and sidebar resizer visuals.

## Fixed

- Fixed sidebar resizers not visually attaching to the panel edges.
- Fixed depth scale, FOV, and ONNX speed controls overflowing outside the right panel at narrow widths.
- Fixed transform-grid inputs exceeding their container bounds when the settings panel is reduced.

- This file is the source summary for future `0.1.1` tags and release notes.
