# 0.1.7

## Added

- Added demo scene orchestration flow for staged scene and camera reveal during editor playback demos.
- Added mesh animation support coverage for imported FBX-style animated assets in the editor pipeline.
- Added regression tests for demo scene orchestration, mesh animation support, playback policy, and orbit roll behavior.

## Changed

- Changed the editor demo flow so scene content can be revealed in a more controlled presentation sequence.
- Changed camera and controller behavior around playback and interaction to better support demo presentation and scene preview.
- Changed orbit and FPS camera control logic to reduce accidental roll-related instability during interaction.

## Fixed

- Fixed several playback and controller edge cases around demo presentation and animated asset handling.
- Fixed editor-side animation integration issues so imported animated meshes behave more consistently in the scene pipeline.

## Notes

- This version closes the demo-play iteration of the editor.
- The main focus of this cycle is demo orchestration, animation playback readiness, and more stable scene-camera interaction.
