# 0.1.2

## Added

- Added scene save/load coverage for more editor state, including visibility, ONNX animation settings, scene camera settings, render mode, and timeline data.
- Added raw-depth capture/readback for fused mesh + gaussian rendering, plus browser debug hooks for raw-depth preview, export, and scene-point inspection.
- Added double-click scene-point look-at based on raw depth picking.

## Changed

- Updated the renderer interface and fused renderer code comments to document the color/normal/depth render flow and where raw depth becomes available.
- Updated the help tips panel to document `Space`, `F`, `Ctrl+F`, and double-click scene-point focus.
- Updated canvas/viewport layout syncing so the render area follows the visible center viewport instead of the full unobscured page canvas.

## Fixed

- Fixed fused renderer initialization ordering for the raw-depth pipeline.
- Fixed ONNX clip/scene-point interaction workflow so double-click focus uses the picked world point while better matching the visible viewport center.

## Notes

- This version is focused on editor interaction quality, scene persistence completeness, and raw-depth-based debugging/picking infrastructure.
