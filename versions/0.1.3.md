# 0.1.3

## Added

- GLB/GLTF 动画加载现在会保留 glTF animation clips，并在编辑器中建立基于 Three.js `AnimationMixer` 的运行时播放封装。
- 为 GLB 动画补充了 OpenSpec 变更文档，包括 proposal、tasks、spec delta 和验证清单。

## Changed

- 编辑器时间线的模型动画逻辑从 ONNX 专用扩展为通用模型动画元数据，GLB/GLTF 动画现已接入变速、clip 区间、loop marker、overflow 提示和场景保存/加载恢复。
- 模型设置面板中的“ONNX 动画”标题改为更通用的“动画”。
- 相机序列可见/隐藏按钮已移动到“相机关键帧”标题右侧，模型列表中不再显示相机序列系统项。

## Fixed

- 修复了 GLB 动画在时间线驱动下未真正激活 `AnimationAction` 导致模型不播放的问题。

## Notes

- 当前仍是 0.1.2 之后的开发中版本，以上内容尚未形成 0.1.3 commit/tag。
