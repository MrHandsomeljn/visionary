# 0.1.4

## Added

- 相机关键帧时间线新增插值方式选择，现支持 `线性`、`Squad`、`Catmull`、`Ease` 四种模式。
- 插值方式右侧新增动态参数控件；`Catmull` 可调张力，`Ease` 可调强度。
- 新增 OpenSpec 变更文档，覆盖相机序列屏幕空间等粗线显示、插值方式选择与多模式扩展。

## Changed

- 相机序列 3D 可视化改为屏幕空间恒定粗细的宽线渲染，近看和远看时线宽保持更稳定。
- `Squad` 模式收敛为更适合相机的行为：优先平滑旋转，避免把自由相机路径拉成不稳定的飞线。
- 相机插值模式与参数的切换现在只更新轨迹与插值配置，不会强制把当前自由相机拉回时间轴当前帧位姿。
- 相机插值模式与参数已接入时间线预览、轨迹采样、导出评估和场景保存/加载恢复。

## Fixed

- 修复相机序列辅助线在不同观察距离下显得过粗或过细的问题。
- 修复 `Squad` 作为相机曲线时导致相机围绕目标异常漂移的问题，拆分为更稳定的多模式插值行为。
- 修复调整插值参数或切换插值模式时，自由相机会被意外拉回当前时间帧相机位置的问题。

## Notes

- 当前为 0.1.4 开发版本汇总，聚焦相机序列可视化与相机插值工作流改进。
