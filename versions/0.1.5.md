# 0.1.5

## Added

- 新增选中模型的视口变换 gizmo，支持移动、旋转、缩放三种模式。
- 新增右侧悬浮工具栏，可直接切换 gizmo 模式并打开悬浮场景设置面板。
- 新增左侧悬浮模型管理面板，支持固定宽度显示、标题栏 `+` 加载模型按钮，以及整面板 `展开 / 收起`。

## Changed

- 模型列表与模型属性合并为画布内左侧悬浮控件，模型属性固定贴底显示，不再占用独立右栏。
- 场景设置改为从右侧工具栏展开的悬浮面板，编辑器整体布局调整为“左侧管理 + 中间视口 + 底部时间轴 + 右侧工具栏”。
- 选中模型后的变换输入、重置操作与视口 gizmo 现在保持双向同步。
- 左侧模型管理区的按钮与面板样式经过压缩和重排，加载、收起、属性编辑的工作流更集中。

## Fixed

- 修复 gizmo 拖拽与相机控制互相抢占的问题，拖拽 gizmo 时会正确抑制相机操作。
- 修复模型数值变换、重置、选择切换后 gizmo 位置不同步的问题。
- 修复左侧模型面板宽度与布局参与主视口挤压的问题，改为固定宽度浮层。

## Notes

- 当前 0.1.5 的主线是“视口直接操控模型”和“编辑器面板重构”。
- OpenSpec 活跃变更 `add-editor-viewport-transform-gizmo` 已完成主体实现，但仍保留一条 mesh / gaussian 实机验证任务，暂不执行正式 archive。
